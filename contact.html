User forum

Users may post questions on the forum. Posting through the web interface requires to log in using a Google account. Alternatively, questions can be posted by sending an email to james-users@googlegroups.com.

Receive updates

Subscribe to the user group to stay informed about ongoing discussions. To subscribe without a Google account, send an email to james-users+subscribe@googlegroups.com using the email address on which you would like to receive the updates. The subject and content can be left blank. To unsubscribe by mail, send a message to james-users+unsubscribe@googlegroups.com.
