Class FastLimitedQueue<E>

  • java.lang.Object
    • org.jamesframework.core.search.algo.tabu.FastLimitedQueue<E>
  • Type Parameters:
    E - type of contained elements

    public class FastLimitedQueue<E>
extends Object
    A fast queue with limited size that replaces the least recently added items if it is full, and which never contains duplicates of the same item. The implementation uses a combination of a queue (to maintain addition order) and a hash set (for fast lookup of elements).
    Author:
    Herman De Beukelaer

        Add an element at the tail of the queue, replacing the least recently added item if the queue is full. The item is only added to the queue if it is not already contained. Runs in constant time, assuming the hash function disperses the elements properly among the buckets of the underlying hash set.
