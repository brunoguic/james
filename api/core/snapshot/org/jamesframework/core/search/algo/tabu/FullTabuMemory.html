Class FullTabuMemory<SolutionType extends Solution>

    All Implemented Interfaces:
    TabuMemory<SolutionType>

    public class FullTabuMemory<SolutionType extends Solution>
extends Object
implements TabuMemory<SolutionType>
    Full tabu memory that stores deeps copies of recently visited solutions and declares a move tabu if applying it to the current solution yields a neighbouring solution which is currently contained in the memory. A full tabu memory has a single parameter controlling the size of the memory, i.e. the number of recently visited solutions which are simultaneously stored. If the size is exceeded, the least recently visited solution is discarded from the memory (FIFO).
