org.jamesframework.core.problems.solutions

Class SubsetSolution

  • java.lang.Object
    • org.jamesframework.core.problems.solutions.Solution
      • org.jamesframework.core.problems.solutions.SubsetSolution

  • public class SubsetSolution
extends Solution
    Implements a generic subset solution based on IDs of selected entities. The subset is sampled from a set of entities which are each assumed to be uniquely identified using an integer ID, so that a generic subset solution can be modeled using these IDs only, independently of the actual corresponding entities.
    Author:
    Herman De Beukelaer

      • SubsetSolution

        public SubsetSolution(Set<Integer> allIDs)
        Creates a new subset solution given the set of all IDs, each corresponding to an underlying entity, from which a subset is to be selected. Initially, no IDs are selected. Note: IDs are copied to the internal data structures of the subset solution; no reference is stored to the set given at construction. IDs are stored in sets that do not guarantee any order. See SubsetSolution(Set, boolean) to create a subset solution that stores IDs in sorted sets.
        Parameters:
        allIDs - set of all IDs from which a subset is to be selected
        Throws:
        NullPointerException - if allIDs is null

      • SubsetSolution

        public SubsetSolution(Set<Integer> allIDs,
              Set<Integer> selectedIDs)
        Creates a new subset solution given the set of all IDs, and the set of currently selected IDs. Note: IDs are copied to the internal data structures of the subset solution; no reference is stored to the sets given at construction. IDs are stored in sets that do not guarantee any order. See SubsetSolution(Set, boolean) and SubsetSolution(Set, Set, boolean) to create subset solutions that store IDs in sorted sets.

        selectedIDs - set of currently selected IDs (subset of all IDs)
        Throws:
        NullPointerException - if allIDs or selectedIDs are null, of selectedIDs contains any null elements
        SolutionModificationException - if selectedIDs is not a subset of allIDs

      • SubsetSolution

        public SubsetSolution(SubsetSolution sol)
        Copy constructor. Creates a new subset solution which is identical to the given solution, but does not have any reference to any data structures contained within the given solution (deep copy). The obtained subset solution will have exactly the same selected/unselected IDs as the given solution, and if IDs are ordered in the given solution this ordering will be retained.
        Parameters:
        sol - solution to copy
