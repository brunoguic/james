James – A Java Metaheuristics Search Framework

James is an extensive Java framework for optimization using local search metaheuristics. The framework is mainly focused on subset selection, with applications in many areas, though other types of problems can also easily be plugged in.

Under Construction

The homepage of the James framework is currently under construction. Please come back later for detailed information and examples on how to use the framework. You can already view the source code on GitHub.
