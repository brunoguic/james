James – A Java Metaheuristics Search Framework

James is an extensive Java framework for optimization using local search metaheuristics. The framework is mainly focused on subset selection, with applications in many areas, but other types of problems can also easily be implemented.

Get started with James

Download and install the latest version of the James framework. You can either add a Maven dependency to automatically pull in all required packages or you can manually download and install the JAR-files.
